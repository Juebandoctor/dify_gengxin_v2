# 安徽省人民政府关于强化国土空间规划基础作用促进“三地一区”建设的意见
各市、县人民政府，省政府各部门、各直属机构：
为全面贯彻党的二十大和二十届二中、三中全会精神，深入贯彻落实习近平总书记考察安徽重要讲话精神，强化国土空间规划基础作用，加快推进“三地一区”建设，提出如下意见。
一、构建支撑“三地一区”发展空间新格局，筑牢可持续发展的空间基础
（一）落实国家重大战略。落实长三角一体化发展、长江经济带发展、中部地区崛起等国家重大战略，推动共建“一带一路”高质量发展，以“三区三线”为基础，围绕“一圈两屏三带五区”国土空间总体格局，统筹发展和安全，优化农业、生态、城镇空间。（牵头单位：省自然资源厅、省发展改革委，各市县人民政府，以下均需各市县人民政府落实，不再列出）

（二）筑牢粮食安全底线。完善耕地占补平衡制度。（牵头单位：省自然资源厅）推进耕地质量提升工程，逐步把具备条件的永久基本农田建成高标准农田。（牵头单位：省农业农村厅）健全“长牙齿”的耕地保护硬措施，确保耕地保有量和永久基本农田保护面积不减少。（牵头单位：省自然资源厅、省农业农村厅）
（三）严守生态安全底线。落实生态环境分区管控制度。推进突出生态环境问题排查整治。（牵头单位：省生态环境厅）加强生态保护红线、自然保护地、湿地管理，开展生态保护红线保护成效评估。（牵头单位：省自然资源厅、省林业局、省生态环境厅）编制实施自然保护地专项规划，推进自然保护地体系建设。提升生物多样性保护水平。强化林地用途管制，实行林地分级保护和占补平衡管理。（牵头单位：省林业局）
（四）保障水资源安全。严格落实用水总量和强度双控制度。加强河湖岸线空间管控，实施水资源战略储备，优化城乡饮用水水源地和备用水源地布局。（牵头单位：省水利厅、省生态环境厅）优化水利基础设施空间布局，保障重大水利工程建设。探索区域水资源调配，推动皖北地区水源置换。（牵头单位：省水利厅、省自然资源厅）

（五）提高国土空间安全韧性。完善淮河、长江干流骨干防洪工程体系，完善皖南山区、大别山区及沿淮区域防洪体系。优化蓄滞洪区范围，蓄滞洪区内严禁建设开发园区、城市新区。（牵头单位：省水利厅）市县编制应急避难场所专项规划。（牵头单位：省应急管理厅）实施搬迁避让优先、搬迁与治理相结合的地质灾害综合治理。（牵头单位：省自然资源厅、省应急管理厅）推进地下水超采和采煤沉陷区综合治理。（牵头单位：省水利厅、省发展改革委、省自然资源厅）强化水土流失综合防治。（牵头单位：省水利厅）加强地震监测预警、灾害风险评估和农村困难群体危房抗震加固改造。（牵头单位：省地震局、省应急管理厅、省住房城乡建设厅）
# 二、聚焦打造科技创新策源地，强化科创空间保障
（六）支撑区域创新空间。坚持生态优先、绿色发展，编制合肥都市圈国土空间规划，超前谋划环巢湖地区空间规划，优化功能布局，为合肥滨湖科学城预留发展空间。推动长三角国家技术创新中心安徽中心布局，与南京、杭州、武汉都市圈等创新空间协同联动，与沪宁、沿江创新带联动，促进高水平创新平台跨区域协作共建。未来大科学城、国家实验室以及其他综合性国家科学中心重点项目建设，可评估后调整国土空间规划。（牵头单位：省自然资源厅、省发展改革委、省科技厅，合肥市人民政府）

# 四、聚焦打造改革开放新高地，促进区域空间协同

# 六、加强规划实施保障
（二十）加强组织领导。各级人民政府要加强组织领导，明确责任分工，健全工作机制，统筹协调解决国土空间保护开发中的重大问题。自然资源部门要会同有关部门协调配合，加强指导、监督和评估，确保实现国土空间规划确定的各项目标任务。省相关主管部门按照工作分工履行职责。（牵头单位：省自然资源厅）
（二十一）健全规划支持政策。落实新增城镇建设用地指标配置同常住人口增加协调机制，使优势地区有更大发展空间。（牵头单位：省自然资源厅、省发展改革委）落实城镇开发边界局部优化制度。完善规划“留白”制度。建立国土空间专项规划目录清单及动态调整机制，自然资源主管部门统筹协调交通、水利、公共管理、公用设施及其他各类专项规划空间布局。探索将生态保护红线内退出建设用地按比例折算纳入城镇开发边界新增城镇建设用地规模机制。（牵头单位：省自然资源厅）
（二十二）强化空间用途管制。完善和落实国土空间用途管制和规划许可制度。实施城乡开发建设、整治更新、保护修复活动应当符合国土空间规划。（牵头单位：省自然资源厅）纳入国土空间总体规划项目清单的交通、水利、能源及其他重点建设项目，在用地范围确定并按程序落入规划“一张图”后，视作符合规划，依法办理用地手续。（牵头单位：省自然资源厅、省发展改革委、省交通运输厅、省水利厅）
